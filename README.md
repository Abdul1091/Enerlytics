# Enerlytics

AI-powered energy intelligence platform for electricity reliability monitoring and grid analytics.

## Overview

Enerlytics enables consumers to report electricity outages, voltage issues, and infrastructure faults while providing utilities and stakeholders with real-time analytics and actionable insights. The platform aims to improve visibility into last-mile electricity service and support data-driven decision-making.

## Features

- Outage reporting
- Voltage quality reporting
- Transformer fault reporting
- Analytics dashboard
- Geospatial visualization
- AI-powered insights (planned)
